# Flight Path Visualization

A Python script that visualizes flight routes and finds the shortest path between cities using Dijkstra's algorithm.

## Example Output

### Original Flight Network
![Original Flight Graph](images/downloa.png)

### Shortest Path Visualization
![Shortest Path](images/download.png)

## Features
- Visualizes flight network as a weighted graph
- Finds shortest path between any two cities
- Highlights the shortest path in red
- Displays distances between cities
